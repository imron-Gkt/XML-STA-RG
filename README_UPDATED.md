# XML-STA-RG: Elite LLM Prompting System

**Elite-level LLM prompts designed for zero-latency execution, precise reasoning, and hallucination control.**

## 🎯 Vision

XML-STA-RG represents the convergence of three powerful paradigms:
- **XML Structures** - Rigid, validatable instruction format
- **STA-RG Framework** - System, Tactical, Analysis, Reasoning, Goal
- **Elite LLM Control** - Production-grade prompt engineering

## 🏗️ Framework Architecture

### The STA-RG Model

```
XML-STA-RG:
├── System (S)      → Role definition & capabilities
├── Tactical (TA)   → Operational execution plan
├── Analysis (A)    → Analytical frameworks
├── Reasoning (R)   → Validation & verification
└── Goal (G)        → Success criteria & delivery
```

### XML Layer Structure

```xml
<system_prompt version="1.0">
  <system>
    <role>Define the AI system's role</role>
    <expertise>Define areas of expertise</expertise>
    <boundaries>Define operational boundaries</boundaries>
  </system>
  
  <tactical>
    <operations>Define operational steps</operations>
    <sequence>Define execution sequence</sequence>
    <fallback>Define fallback procedures</fallback>
  </tactical>
  
  <analysis>
    <framework>Define analytical approach</framework>
    <methodology>Define evaluation method</methodology>
    <criteria>Define success criteria</criteria>
  </analysis>
  
  <reasoning>
    <validation>Define validation logic</validation>
    <verification>Define verification steps</verification>
    <gates>Define quality gates</gates>
  </reasoning>
  
  <goal>
    <primary>Define primary objective</primary>
    <secondary>Define secondary objectives</secondary>
    <success_metrics>Define success metrics</success_metrics>
    <output_format>Define output format</output_format>
  </goal>
</system_prompt>
```
